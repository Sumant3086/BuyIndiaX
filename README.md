# BuyIndiaX - MERN E-Commerce

## Quick Start

**Local:**
```bash
npm install && cd client && npm install && cd ..
npm start                    # Backend (5000)
cd client && npm start       # Frontend (3000)
```

**Deploy to AWS:**
```bash
cd terraform && terraform apply -auto-approve
```
