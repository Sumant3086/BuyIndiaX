BuyIndiaX - Payment Integration Guide
=====================================

PAYMENT METHOD CONFIGURED:
-------------------------
✅ Razorpay Payment Link: https://razorpay.me/@sumantyadav

HOW IT WORKS:
------------
1. Customer completes checkout with shipping address
2. Order is created in the database
3. Customer is redirected to your Razorpay payment link
4. After payment, customer returns and enters Transaction ID
5. Payment is confirmed and order status is updated

PAYMENT FLOW:
------------
Checkout → Create Order → Open Payment Link → Complete Payment → 
Enter Transaction ID → Confirm Payment → Order Complete!

ADVANTAGES:
----------
✅ No API keys needed - uses your existing payment link
✅ Simple integration
✅ Customers can pay via UPI, Cards, Net Banking
✅ Manual confirmation ensures payment verification
✅ Works immediately without additional setup

OPTIONAL - FULL RAZORPAY API INTEGRATION:
-----------------------------------------
If you want automated payment verification:

1. Sign up at https://razorpay.com/
2. Go to Settings → API Keys
3. Generate Test/Live API Keys
4. Update .env file:
   RAZORPAY_KEY_ID=rzp_test_xxxxx
   RAZORPAY_KEY_SECRET=xxxxx

With API keys, payments will be automatically verified without
manual transaction ID entry.

TESTING PAYMENTS:
